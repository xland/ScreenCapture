# ScreenCapture

Мощный и компактный инструмент для создания скриншотов в Windows.

## Возможности

- Скриншоты, аннотирование, скроллинг-скриншоты (длинные снимки), запись экрана.
- Пипетка с горячими клавишами для копирования цветов RGB (`Ctrl+R`), HEX (`Ctrl+H`) и CMYK (`Ctrl+K`).
- Рисование заполненных и незаполненных эллипсов, идеальных кругов (удерживайте `Shift`), прямоугольников, квадратов (удерживайте `Shift`), стрелок, нумерованных меток и т.д.
- Рисование кривых, прямых линий (удерживайте `Shift`), мозаики, ластика, текста.
- Изменение или удаление нарисованных элементов в любое время (наведите мышь на элемент).
- Отмена (`Ctrl+Z`), повтор (`Ctrl+Y`), сохранение в файл (`Ctrl+S`), сохранение в буфер обмена (`Ctrl+C` или двойной щелчок).
- Скроллинг-скриншот (длинный снимок), запись экрана (сохранение в форматах GIF и MP4).
- Высокая скорость работы при низком потреблении памяти.
- Компактный размер, один исполняемый файл, установка не требуется, не требует сторонних DLL.
- Поддерживает запуск указанных функций напрямую через аргументы командной строки.
- Поддерживает режим однократного запуска (процесс не остаётся работать в памяти после завершения).

## Поддерживаемые операционные системы

- Windows 10 1607 или новее

## Компиляция

- Этот проект не имеет зависимостей от сторонних библиотек, кроме библиотек, встроенных в операционную систему.
- Проект может быть скомпилирован с помощью Visual Studio 2026 (с установленным набором инструментов для разработки на C++).

## Командная строка

- Не добавляйте пробелы по обе стороны от знака равенства в аргументах командной строки.
- Три типа аргументов ниже можно комбинировать. Пример:

```
> ScreenCapture.exe enter=long tray=false auto-quit=true
```

```
// Запустить программу и сразу перейти в режим скриншота (по умолчанию).
> ScreenCapture.exe enter=cap

// Запустить программу и перейти в режим длинного скроллинг-скриншота.
> ScreenCapture.exe enter=long

// Запустить программу и перейти в режим записи экрана.
> ScreenCapture.exe enter=video
```

```
// Показать значок в системном трее (по умолчанию).
> ScreenCapture.exe tray=true

// Скрыть значок в системном трее.
> ScreenCapture.exe tray=false
```

```
// Оставить процесс запущенным после создания скриншота.
> auto-quit=false

// Завершить процесс сразу после создания скриншота.
> auto-quit=true
```
